# TutorManagementModule
A Console-Based Tutor Management Program using *Doubly Linked List* 

Functionality: 
  1. Create Tutor
  2. Remove Tutor
  3. Find Tutor
  4. Amend Tutor
  5. List All Tutors
  6. Filter Tutors Based on Criteria
     - By Rating
     - By Faculty
  7. Report Generation
     - Top 5 Rated Tutors
     - Faculty Summary Report
  8. Commits Changes to File
